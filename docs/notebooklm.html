NotebookLM → JSON

Nueva feature

Transforma la salida estructurada de NotebookLM y la convierte en JSON de examen

Elige un formato simple para que NotebookLM responda con bloques de preguntas. Después pega esa salida aquí y la herramienta la convierte en un examen compatible.

1. Preset del examen

El preset rellena automáticamente asignatura, convocatoria, número de preguntas, penalización y tiempo.

2. Salida de NotebookLM

Pega directamente el JSON o el bloque envuelto por fences que devuelve NotebookLM. El resultado final será un examen compatible con el visor.
Tip: ver formato de JSON recomendado 
[
  {
    "id": 1,
    "tema": "Tema 6: Organización General del Sistema Nervioso",
    "pregunta": "En relación a la plasticidad neural podemos decir que:",
    "opciones": {
      "a": "El aprendizaje constante en etapas tempranas potencia nuevas conexiones.",
      "b": "Los astrocitos participan en la plasticidad sináptica.",
      "c": "Pueden cambiar el número de espinas dendríticas.",
      "d": "Todas las anteriores son correctas."
    },
    "correcta": "d",
    "explicacion": "La plasticidad neural implica cambios estructurales y funcionales relacionados con aprendizaje y memoria."
  },
  {
    "id": 2,
    "tema": "Tema 1",
    "pregunta": "Texto de la pregunta...",
    "opciones": {
      "a": "Opción A",
      "b": "Opción B",
      "c": "Opción C",
      "d": "Opción D"
    },
    "correcta": "b",
    "explicacion": "Justificación breve."
  }
]

3. Parámetros del examen

Sin carpeta de catálogo elegida. Selecciona localmente docs/assets/json para publicar y regenerar el índice.
